# Vibe-Coding-Studio
A knowledge base for AI coding tools, including settings, skills, agent definitions, and workflow templates.


## Purpose

This repository helps keep useful configuration in one place for:
- Claude Code
- OpenCode
- Antigravity

It is designed to store:
- app settings
- reusable Skills files
- `Claude.md` instructions
- agent and subagent definitions
- workflow notes
- best practices and setup examples

## What is included

- App-specific configuration notes
- Reusable Skills templates
- Agent and subagent templates
- Prompt and instruction patterns
- Setup examples for different coding workflows

## Principles

- Keep instructions short and clear.
- Prefer reusable templates over one-off prompts.
- Use consistent naming across tools.
- Store only settings that are actually useful.
- Make files easy to scan and update.

## Recommended content to save

- system prompts
- project rules
- coding style preferences
- agent roles
- subagent responsibilities
- tool usage rules
- debugging habits
- review checklists
- quality standards
